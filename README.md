# RideShare 

This is our mini project for oops lab.

## How to clone this repository?

1. Make a folder called RideShareProject (or whatever name you like). 

2. Open it in vs code (or your preffered IDE).

3. Open terminal and run.
   ```bash
      git clone https://github.com/SuPriya112K/oops-lab-project-3sem.git
   ```   

## How to run this project on your local machine?

1. Compile all java files.
   ```bash
      javac RideShare/*.java
   ```

2. Run the Main file.
   ```bash
      java RideShare.Main
   ```
